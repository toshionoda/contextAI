現在のセッションでは Notion MCP サーバー（`claude_ai_Notion`）が接続されていません。

過去のセッションでは使えていた履歴があります（`.claude/settings.local.json` の allow list に記録済み）が、今回のセッションでは MCP ツールが起動していない状態です。

**考えられる原因と対処法：**

1. **claude.ai 統合の再接続が必要** — `mcp__claude_ai_Notion__*` という名前空間は claude.ai プラットフォームのネイティブ統合です。Claude Code を再起動するか、以下を確認してください：
   ```
   claude mcp list
   ```

2. **接続確認** — Claude Code のターミナルで次を実行：
   ```
   /mcp
   ```
   または Settings → MCP で Notion の接続状態を確認

3. **再接続後に再実行** — 接続が復活したら、同じプロンプトをもう一度送ってください。すぐにサマリーを生成します。

Notion MCP が使えるセッションで実行した場合、手順（`notion-query-meeting-notes` → `notion-search` × 4 → `notion-fetch` で詳細取得 → Markdown サマリー出力）はすぐに動かせる状態です。
