**04/17 16:33** (U083Y29495F): CiviLinkで掲示板のコメント通知メールが届かない事象について続報です。

■ 原因
4/15 の Repository 層リファクタリング（#2242）で RepositoryUtil.create のデフォルトが auto_commit=True → False に変更された影響。
Notification テーブルの UUID 系カラム（String(256)）に Python UUID オブジェクトがそのまま残り、
通知モデル生成時に型エラー（`AttributeError: 'UUID' object has no attribute 'replace'`）が発生していました。

■ ユーザー向けの原因
4月15日のシステム内部改修において、通知データの処理方式が変更されました。
その際、掲示板スレッドの通知に関わるデータの受け渡し部分で不整合が発生し、通知の生成処理が正常に完了しない状態になっていました。

■ 発生期間
2026年4月15日〜現在にかけて
