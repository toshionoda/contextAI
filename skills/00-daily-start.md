# 00 - 始業チェック

## 目的
朝の状況把握・当日の優先度を整理する。

## 実行タイミング
- 毎朝始業時

## 手順

### 1. Slack確認（直近の動き）
- `context/slack/` から直近のメンション・顧客チャンネルの動きを確認
- 緊急対応が必要なものをピックアップ

### 2. カレンダー確認
- `context/calendar/` から本日のMTGを確認
- MTGがある場合は `06-meeting-prep.md` を実行

### 3. 顧客アラート確認
- `customers/*/README.md` のヘルススコアを確認
- 🔴 Critical の顧客は最優先で対応

### 4. タスク確認・優先度設定
- `tasks/todo.md` を確認
- 当日やるべきことを「📅 今週」セクションに整理

### 5. 日次サマリー生成
- `context/daily/YYYY-MM-DD_summary.md` を生成

## 出力
- 当日の優先タスクリスト
- 顧客アラートサマリー
- MTG予定の確認
