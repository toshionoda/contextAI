# 02 - オンボーディング支援

## 目的
新規顧客のCiviLink導入をスムーズに進め、早期定着を実現する。

## 実行タイミング
- 新規顧客の導入開始時

## 手順

### 1. 顧客ディレクトリ作成
- `customers/_template/` を複製
- 基本情報を記入
- ステータスを 🔵 Onboarding に設定

### 2. 環境ヒアリング
- 図面形式（PDF / DocuWorks / CAD）
- 現在のワークフロー
- 利用目的（図面管理 / 照査 / その他）
- ユーザー数・組織構成

### 3. 導入プラン作成
- Phase 1: キーユーザーによるトライアル（1-2週間）
- Phase 2: チーム展開（2-4週間）
- Phase 3: 全体展開・定着化

### 4. 初期設定サポート
- アカウント設定
- プロジェクト作成
- サンプル図面のアップロード・照査体験

### 5. フォローアップ設定
- 1週間後: 利用状況確認
- 2週間後: 課題ヒアリング
- 1ヶ月後: ヘルスチェック実施

## 注意点
- DocuWorksユーザーの場合、ベクター表示の制約を事前に説明
- PDF運用を推奨するが、既存ワークフローへの配慮も必要
