# 自治体向けCiviLink導入支援プロジェクト

**キックオフ:** 2026-04-09
**メンバー:** 高取 佑 / 長谷川 健一 / 野田 敏雄
**パートナー:** 伊藤忠商事（佐賀市包括委託スキーム）

---

## 方針

佐賀市を最優先ターゲットとし、包括委託開始が1年延期された期間をPOCの場として活用する。
地場建設業者へのDX普及推進を軸に、CiviLinkをプラットフォームとして根付かせ、施工会社→自治体の順でアプローチする。

## ターゲット自治体

| 自治体 | ステータス | 優先度 | 備考 |
|--------|-----------|--------|------|
| [佐賀市](saga/) | POC準備中 | **最優先** | 伊藤忠経由。4/20訪問予定。デジタル推進課所管 |
| [神戸市](kobe/) | 接続済み（不確定） | 中 | ミートアップ経由 |
| [北九州市](kitakyushu/) | 調査済み | 低（個人ネットワーク） | 大学同期経由でコンタクト予定 |

## 戦略

1. **短期（〜2026年4月）**: 4/20佐賀市訪問に向けたDX支援メニュー資料準備
2. **中期（〜2027年3月）**: 佐賀市でのPOC実施、実績づくり
3. **長期**: 佐賀市の実績を足がかりに全国自治体へ横展開

## キーポイント（2026-04-13 田部井・しなべMTGで更新）

- **CiviLinkの位置づけ**: データプラットフォーム → **「既存ツールのオーケストレーター」**（既存ツール間のAPI連携・データ統合の中核）
- **マルメの当面の役割**: CiviLink提供者ではなく**DXコンサル事業会社**として、既存ツールをパッケージ提案
- **対象企業**: 佐賀市契約の**Cランク以下の地元建設会社**（IT/DX担当者が少なくアナログな現場）
- **対象分野**: **橋梁の維持管理・補修設計・補修工事**
- **提案アプローチ**: 既存ICTツール（チャット・クラウド・写真アプリ・Webカメラ等）をパッケージで導入 → CiviLinkでデータ連携
- **書類DX**: 自治体⇔請負業者間の契約書・請求書の統一書式化が刺さる領域
- **CiviLink差別化**: 異なるツールから集まるデータをAIで横串活用するオーケストレーション機能
- 赤字/無償でのPOCも許容（全国展開の足がかりとして重要）
