## 🥇 Tier 1：今週〜2週間で必ず取る（Go判断の必須インプット）

### 1. mirai@DX担当者（**起点・元下水道部署**）

- **位置づけ**: 既に繋がっている起点人物。本人がチャンピオン候補であり、元下水道部署のネットワーク保持者
- **ヒアリング目的**: チャンピオンの本気度確認＋紹介ハブとしての役割確定
- **接触方法**: Slack DM → 30分オンライン
- **必須質問**:
  - 「製品があったら使いたい」と言ってくれた**元下水道部署の具体的な部署名・キーパーソン**は誰か？（→ 質問#2の相手の特定）
  - mirai@の**実証実験プログラムでPoC経費を出せる枠は存在するか**（採択枠・予算規模・申請時期）
  - 福岡市の**スタートアップ公共調達サポート事業**で随意契約可能なルートはあるか
  - **東芝CISSART・他社AI積算ベンダーの接触状況**（営業を受けたことがあるか／RFI出したか）
  - mirai@担当者自身が**どこまで旗を振れるか**（紹介止まり／同行可／予算化提案までできるか）
- **期待アウトプット**: 質問#2の対象者紹介、PoC経費の有無、競合状況

### 2. 道路下水道局 元下水道部署のキーパーソン（**当事者**）

- **位置づけ**: **痛みの当事者**。mirai@担当者の前任ライン
- **ヒアリング目的**: 業務時間の定量、現行積算ソフトの実態、「使いたい」の本気度
- **接触方法**: mirai@紹介 → 対面30-60分（局舎訪問）
- **必須質問（Need定量）**:
  - 下水道工事の**年間発注件数**と1件あたりの**積算工数**（実数値、推定でよい）
  - 現在使っている積算ソフト（**積算王／APRO／ARES／頂／ランド2000**のどれか）と、図面読取は完全人手か
  - **数量計算書ドラフト作成にかかる時間／週**（最大の痛みポイントの特定）
  - **整合性チェック差し戻し**で発生する手戻り件数・時間
  - 委託発注している場合の**外注費**（人件費換算でいくら）
- **必須質問（Authority/Champion）**:
  - この製品を**現場として導入したい立場として、上申先**はどこか（課長 → 部長 → 局長 → 副市長？）
  - 局内で**「便利だけど使わない」を生む障害**は何か（既存ベンダーとの関係・調達ルール・組織政治）
- **期待アウトプット**: SAM試算の精度向上、Champion確定、Need定量化

### 3. 道路下水道局 経営企画部（または下水道経営企画課）

- **位置づけ**: **予算・調達の決裁ライン**。Authority/Budget/Procurementの中核
- **ヒアリング目的**: 実調達ルートの存在確認
- **接触方法**: 質問#2のChampionから紹介してもらう or 直接アポ（30-45分）
- **必須質問**:
  - DX関連ツール導入の**令和8〜9年度予算**の検討状況（要求時期：通常6-9月）
  - **使用料/SaaS型契約**の前例（年額契約・複数年契約の可否）
  - **随意契約**または**少額随契**で導入可能な閾値（福岡市の場合いくらまでか）
  - 上下水道局との**予算分担**（道路下水道局の単独予算で動けるか）
  - **PoC無償提供の場合の本採用パス**（「無償で使ってよかったから来年予算化」が成立する組織か）
- **期待アウトプット**: 価格設定の上限・下限、調達ルート確定

## 🥈 Tier 2：Go判断後すぐ（Phase 1 移行前に固める）

### 4. 福岡市総務企画局 DX戦略課

- **位置づけ**: 全市DX予算の上位枠。mirai@事業の所管
- **ヒアリング目的**: 横断予算で乗れる枠があるか
- **必須質問**:
  - **庁内DX推進事業**でAI積算系の事業計上枠はあるか
  - **mirai@経費負担**の判断権限（mirai@担当者個人 vs DX戦略課決裁）
  - 政令市DX連絡会で**他政令市の同種事例**（横浜市・名古屋市・大阪市の上下水道DX）の情報共有はあるか
- **期待アウトプット**: 横展開時の他政令市紹介ルート

### 5. 福岡市上下水道局（**水道側**）

- **位置づけ**: 道路下水道局とは別組織。**水道工事の数量計算**も同様の課題を持つ可能性
- **ヒアリング目的**: スコープ拡張余地・販売対象のダブル化
- **必須質問**:
  - 水道管布設替工事でも**同じ積算課題**を抱えているか
  - 上下水道局単独で**予算化できる別ライン**があるか
  - 道路下水道局との**ツール共同採用**の前例
- **期待アウトプット**: 1件取れた時のARR倍増可能性

### 6. 既存契約ベンダー1社（積算王/APRO/ARES等の福岡市担当）

- **位置づけ**: 競合になりうる既存接点。福岡市内の意思決定構造を熟知
- **ヒアリング目的**: 福岡市の調達文化・既存ベンダーロイヤリティの強さ
- **接触経路**: 中立的に「業界動向ヒアリング」として接触（直接競合宣戦は避ける）
- **必須質問**:
  - 福岡市での**契約形態**（単年/複数年・保守料込み/別）
  - 数量算出の**A段階自動化**を自社ロードマップに持つか
  - **東芝CISSARTの福岡市接触**の有無
- **期待アウトプット**: 競合動向＋既存ベンダーロックインの強度

## 確度マトリクス（ヒアリング後の集計用）

```
        高 │ ┌─────────┬─────────┐
   B(予算) │ │ 様子見   │ Go確定  │
        中 │ │          │ 即Phase1│
        低 │ └─────────┴─────────┘
            低         高
              N(痛み定量×Champion強度)
```

判定基準:
- **右上（Go確定）**: 予算化見込み70%以上 × Champion明確 → Phase 1 即着手
- **右下（即Phase1）**: 予算化見込み低いがNeed・Champion強 → 無償PoC継続で来年度予算化を狙う
- **左上（様子見）**: 予算はあるがNeed弱／Champion不在 → ヒアリング追加
- **左下**: No-Go
