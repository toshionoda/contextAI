- 共通仮設費（管内洗浄工 / 本管TV調査工 / 障害物撤去工 など）
- 管布設工 / 管路土工（掘削・埋戻・残土処分）
- 取付管およびます工（小口径汚水桝・取付管布設工・やりとり継手）

これら除外項目は計算書PDFに記載されていても **無視する**。L2が「管きょ更生工」以外のものは出力しないこと。

## 1. 工事の種類

下水道改築工事のうち、**管更生工**（既設管渠の内面更生）を主体とする工事を対象とする。
新設工事（開削・推進・シールド）は対象外（Phase 1 MVP で対応）。

## 2. 路線の表記ルール

- 路線番号: `429-2`, `430-3`, `410-1-3`, `86-1`, `209-4`, `452-5` 等のハイフン区切り数字
- 図面では路線番号 + 管径 + 工種 + 荷重区分 + 施工時間帯 が一括表記される
  - 例: `429-2 / TPφ250 / 管更生 / T-14 / レベル1 / 昼間施工`
- 数量計算書では「整理番号 + 路線番号 + 路線」の形式
  - 例: `整理番号8 / 86-1 路線`

## 3. 管種・管径の表記

| 表記 | 意味 |
|---|---|
| `TPφ250` / `TPφ300` | 既設陶管 |
| `VUφ250` / `VUφ150` | 硬質塩化ビニル管 |
| `φ250` / `φ300` / `φ150` | 一般的な内径表記（mm） |
| `Tφ150` | 取付管（陶管 φ150） |
| `T管` | 陶管 |

呼び径レンジ：φ150（取付管）/ φ200, φ250, φ300（本管）/ φ400+（大口径） が標準。

## 4. 工種・種別・細別（4階層構造）

数量総括表は **L2(工種) → L3(種別) → L4(細別) → 規格 → 単位 → 数量** の階層。
管更生工事の主要4分類：

### 4.1 管きょ更生工
- **更生管材**: 単位 `m`、規格 `更生自立管 既設管渠φ250` など
- **内面更生工**:
  - 反転・引込工
  - 硬化・形成工
- **更生付帯工**:
  - 仮設備設置・撤去工（単位 `回`）
  - 本管口切断工
  - 本管口仕上工（単位 `箇所`、両端ごと）
  - 取付管口せん孔仕上工（単位 `箇所`、取付管数）
- **換気工**: 換気設備工（単位 `式`）
- **管きょ更生水替工**: 揚泥車運転工 / 反転・形成水替（単位 `式`）

### 4.2 管布設工（部分布設替えがある場合のみ）
- **管路土工**: 掘削 / 管路埋戻 再生砂 / 運搬・処分（単位 `m3`）
- **管布設工**: 管布設工 VU管φ250 / マンホール用継手（上流用・下流用） / 異種管継手 / 埋設テープ
- **管路路面覆工**: しま鋼板 2.4×1.2
- **管路土留工**: 矢板工 H=2.0m/2.5m/3.0m / 土留支保工 1段・2段

### 4.3 取付管およびます工・マンホール工
- **取付管およびます工**:
  - ます設置工 小口径汚水桝 T-14 / T-25（単位 `箇所`）
  - 取付管布設工: 取付管および支管取付工 3m未満 / 3m以上から5m未満
  - やりとり継手 φ150（単位 `個`）
  - 標示テープ 巾50mm 2回巻
  - 管路土工: 床掘 / 管路埋戻 再生砂 / 運搬・処分
  - 管路土留工: 矢板工 / 土留支保工
- **マンホール工**:
  - 人孔改築工:
    - 鋳鉄製マンホール蓋 φ600 T-14 / φ600 T-25（単位 `組`）
    - 人孔口環 h=100 / h=150（単位 `個`）
    - 足掛金物設置工 W=300（ステンレス製、単位 `本`）
    - 転落防止梯子（単位 `組`）
    - 流動性特殊モルタル t=20-70mm（単位 `m3`）
    - インバート工 1号人孔（単位 `箇所`）
  - 人孔土工: 掘削 / 埋戻 / 運搬・処分

### 4.4 付帯工
- **舗装撤去工**:
  - 舗装版切断 t=15cm以下 / t=15cm-30cm（単位 `m`）
  - 汚泥運搬処分（単位 `m3`）
  - 舗装版破砕積込 t=5cm以下 / t=15cm以下（単位 `m2`）
  - インターロッキング撤去
  - 殻運搬処分 As / Co（単位 `m3`）
  - 掘削 / 運搬・処分（単位 `m3`）
- **舗装復旧工**: 4種類（A/B/C/D）+ 歩道
  - 仮復旧工: 仮復旧A / B / C / D / 歩道
  - 本復旧工: 仮〜本復旧A / B / C / D / 歩道
  - 影響部A / B / C / D / 歩道（単位 `m2`）
- **区画線設置工**: W=15cm / 実線 W=15cm換算 / 文字・記号 W=15cm換算 / 横断歩道 W=45cm
- **既設構造物撤去工**: 構造物とりこわし 無筋Co / 殻運搬・処分 無筋Co
- **管撤去工**: 既設管撤去 陶管 φ150 / φ250 / 陶磁器くず運搬 / 閉塞キャップ φ250-φ500 / φ600-φ1500
- **スクラップ費**（単位 `t`）

## 5. 共通仮設費（路線横断で記載）
- 準備費:
  - 管内洗浄工 施工前/施工後（単位 `m`）
  - 本管TV調査工 施工前/施工後（単位 `m`）
  - 障害物撤去工: モルタル / 木の根 / 取付管突出 / その他

## 6. 図面の構成と記載項目

下水道工事の図面は以下の構成（JS下水道施設CAD製図基準＋名古屋市完成図作成基準ベース）：

| 図面種別 | 縮尺 | 記載項目 |
|---|---|---|
| 全体位置図 | A1: 1/2,500 / A3: 1/5,000 | 行政区名・町名・主要公共施設・水準基標・方位・縮尺・工事路線 |
| 平面図 | A1: 1/300 / A3: 1/600 | 道路地形・既設管路・新設管路・人孔位置・地下埋設物（水道管・ガス管） |
| 縦断図 | V=1/100 / H=1/300〜1/500 | 上流管底高・下流管底高・勾配 i / 延長 L / 管径 / 管種 |
| 横断図 | A1: 1/100 / A3: 1/200 | 道路幅員・舗装厚 / 埋設深度 |

### 図面の数値抽出時に注意するパターン

- **路線番号**: 平面図上に楕円で囲まれて記載される（例: `429-2`）
- **管種・管径**: 「TPφ250」のような連結表記（管種記号 + φ + 内径）
- **勾配**: `i=11.2‰` 形式（小数点以下1桁、‰=パーミル）
- **延長**: `L=18.10m` 形式
- **管底高**: `9.029` のような小数点3桁（mGL or 標高）
- **荷重区分**: T-14 / T-25（T=トラック、数字=重量t）
- **施工時間帯**: 昼間施工 / 夜間施工
- **マンホール番号**: `00211A-34` のような英数字記号

## 7. 単価コード体系（福岡市独自）

金抜き設計書では以下の単価コードが使われる（Phase C で辞書化）：

| 接頭辞 | 意味 | 例 |
|---|---|---|
| `PG` | 標準歩掛系（市場単価以前のコア工種） | PG64107 換気設備工 / PG15348 ます設置工 / PG15349 取付管布設工 |
| `CB` | 共通単価系（土工・運搬等） | CB210030 床掘り / CB210020 積込 / CB210110 土砂等運搬 / CB430510 舗装版切断 |
| `ZD` | 材料費系（製品・資材） | ZD0203501 再生砂 / ZD0807201 流動性特殊モルタル |
| `PD` | 処分料系 | PD20001 処理（処分）料 / PD35041 構造物とりこわし |
| `ZG` | 部品単価系 | ZG1703001 やりとり継手 / ZG6505002 埋設標示テープ |
| `第N号代価表` | 代価表参照 | 第1号〜第14号代価表（複数歩掛の組合せ単価） |

設計書番号の形式: `B07GH-00-00-G0`（B=下水道, 07=年度, GH=工事区分, ハイフン区切り）

## 8. 検出時のヒューリスティック

### 路線の判別
1. 数量計算書の「整理番号N + 路線番号-X 路線」記載から路線特定
2. 図面の表題欄・吹き出し（楕円）から路線番号確認
3. 縦断図の延長Lと数量総括表の「管きょ延長」を突合

### 数値の解釈
- 延長は **m** 単位（小数点2桁）
- 路線延長 ≠ 管きょ延長（延長は管路センター長、管きょ延長は内面更生対象長）
- 管きょ延長の方が短い（マンホール内寸が引かれるため、約 0.9m / 1人孔）
- 取付管口せん孔の本数 = 該当路線の取付管本数（マンホール間で集計）

### 工種マッチング
- 計算書の工種 L2/L3/L4 → 図面側で対応箇所をマッチング
  - `管きょ更生工 / 更生管材 / -` → 図面の管路（路線記号 + 延長 L）
  - `取付管およびます工 / ます設置工 / 小口径汚水桝 T-14` → 図面の汚水桝記号
  - `付帯工 / 舗装復旧工 / 仮復旧A` → 道路面積（横断図 + 平面図から推定）

## 9. 既知の特殊性（鉄筋PoCとの違い）

| 観点 | 鉄筋PoC（U型擁壁等） | 下水道PoC（管更生工） |
|---|---|---|
| 検出対象 | 鉄筋シンボル（W7, F3 等） | 数値（延長L・管径φ・本数） |
| 構造の次元 | 2D 構造物（断面図） | 1D 線状構造（路線） |
| 規格表記 | D13, D16, D19, D22, D25 | φ150, φ250, φ300, T-14, T-25 |
| 集計の主体 | 1ブロック内の本数 | 路線×工種ごとの延長・本数 |
| 計算書の役割 | 配置鉄筋量照査表 | 数量計算書（路線別計算過程） |
| 図面と計算書の対応 | 配筋詳細図と本数表 | 平面・縦断・横断と数量総括 |

## 10. プロンプト指示（Gemini Vision に渡す要旨）

PDFを画像として処理する場合、以下を遵守:

- **路線番号を最優先で特定**（不明なら "unknown" を返す）
- **数値は計算書記載通り**に出力（丸めない、単位を保持）
- **ハイフン記号は半角ハイフン `-`** で統一
- **見つからない項目は省略**（推測しない）
- **L2/L3/L4 階層は §4 の体系に従う**
