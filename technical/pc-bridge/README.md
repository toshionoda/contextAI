# PC橋設計知識

## 概要
PC橋（プレストレストコンクリート橋）は、コンクリートにあらかじめ圧縮力（プレストレス）を導入した橋梁。

## CiviLinkとの関連
- PC橋設計では大量の図面（一般図、配筋図、緊張図等）が発生
- 照査の効率化がCiviLinkの価値提供ポイント
- JRC等の建設コンサルタントがPC橋設計を多く手がける

## 主な図面種類
- 一般図（側面図、平面図、断面図）
- 配筋図
- 緊張図（PC鋼材の配置）
- 支承図
- 仮設図

## 知っておくべき用語
- プレストレス: コンクリートに事前に導入する圧縮力
- PC鋼材: プレストレスを導入するための鋼材（ストランド、鋼棒等）
- 緊張管理: PC鋼材の緊張力を管理する作業
- 照査: 設計図面の確認・チェック作業
