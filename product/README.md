# product/ - プロダクト知識

CiviLinkに関するプロダクト知識を集約するディレクトリ。

## 構成
- `features/` - 機能一覧・仕様
- `release-notes/` - リリースノート
- `known-issues/` - 既知の課題
- `roadmap.md` - ロードマップ
- `competitive/` - 競合情報

## 更新タイミング
- リリース時: `release-notes/` と `features/` を更新
- 課題発見時: `known-issues/` に追加
- 四半期: `roadmap.md` と `competitive/` を更新
